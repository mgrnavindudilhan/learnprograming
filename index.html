HTML හැැඳින්වීම

HTML (HyperText Markup Language) යනු වෙබ් පිටු සෑදීම සඳහා භාවිතා කරන මූලික භාෂාවයි. එය ටැග් (tags) භාවිතා කරමින් අන්තර්ගතය ව්යුහගත කරයි.

<!-- මෙය HTML අදහස් දැක්වීමකි -->
<h1>මගේ පළමු වෙබ් පිටුව</h1>
<p>මෙය අප්‍රෙල් මාසයේ සිදු වන සිදුවීම් පිළිබඳ විස්තරයකි.</p>
මෙය උත්සාහ කරන්න

මූලික HTML ටැග්

HTML ටැග් යුගල වශයෙන් යෙදේ: ආරම්භක ටැග් සහ අවසන් ටැග්. අවසන් ටැග් තුළ "/" ලකුණ ඇතුළත් වේ.

සුලභ ටැග් කිහිපයක්:

  • <html> - HTML ලේඛනයේ මූල අංගය
  • <head> - මෙටාදත්ත අඩංගු කරයි
  • <title> - බ්‍රව්සර් ටැබ් එකේ පෙන්වන නම
  • <body> - දෘශ්ය අන්තර්ගතය අඩංගු කරයි
  • <h1> - ප්‍රධාන ශීර්ෂය
  • <p> - ඡේදය
ටැග් ගැන වැඩිදුර ඉගෙන ගන්න

HTML පිටු ව්යුහය

සෑම HTML ලේඛනයක්ම මූලික ව්යුහයක් අනුගමනය කරයි:

<!DOCTYPE html>
<html>
<head>
  <title>පිටුවේ මාතෘකාව</title>
</head>
<body>

  <h1>මගේ පළමු ශීර්ෂය</h1>
  <p>මගේ පළමු ඡේදය.</p>

</body>
</html>
